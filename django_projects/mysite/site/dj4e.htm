HOT-BOT-POL-POT

hot-bot-pol-pot is a moderation discord bot using enhanced-dpy library with many features including:

  • Messages members on join/exit and adds a specified role(s) to them.
  • Server and User info embeds that are very informative.
  • Cool Logging Features spanning through member updates (avatar changes, nickname, username, denominator), message updates (edit/delete).
  • Lots of Modding Commands like ban-unban-kick-mute-unmute-add/remove roles and dming members, purging messages and sending messages to other channels.
  • ModMail for communicating between the mod team and server members.
  • User can post Anonymous Confessions with their ids hashed and mods have the ability to block users from confessing using their confession id.
  • Mod Related Logs for kicks - bans - dming and a lot more.
  • Informative Error Messages for a lot of command errors.
  • Fun Commands like fetching wallpapers from unsplash, checking the weather, rolling dice, slapping other members, polls, kanye quotes, ligma detector .
  • Lots of Fancy Embeds for logs and a lot of other Messages with paginated embeds for the help command.

--------------------------------------

Installation:

  1. clone repository
  2. put your token, unsplash app_id, mongodb password, openweathermap app_id in your system environment variables.
  3. change your mongodb username in src/data_clusters/__init__.py
  4. configure the bot using config.py or from the server
  5. run python -m hot-bot_main
